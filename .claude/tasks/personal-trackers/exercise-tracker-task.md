# Exercise Tracker

## What it does
Log workouts and view exercise history.

## Features
- Add an exercise (type, name, duration, optional details)
- View all exercises
- View summary grouped by type
- Delete an exercise

## Exercise types
- cardio (optional: distance in km)
- strength (optional: sets, reps, weight in kg)
- flexibility
- sports
- other

## Color theme of the html page 
Teal and cream
