# Claude Beyond Prompting

Most of us know how to prompt Claude (or any chatbot). But what's the next step beyond prompting?

This repo answers that — one small lesson at a time, each with a video and something you build yourself.

## Lesson 1: Your first agentic session

We take a small, real task and go from prompting to *delegating*:

1. **Pick a small deliverable** — something concrete Claude will build for us
2. **Create a `CLAUDE.md`** — the project instructions Claude reads every session
3. **Write a task file** — and understand how context actually works
4. **Start a Claude Code session** and prompt it to do the job
5. **Toggle manual vs. auto mode** — and learn to steer the work while it runs
