# Personal Trackers Project

This project contains personal trackers built in Python as HTML tools.

## Trackers

- **Expense Tracker** - Track daily expenses by category (food, transport, utilities, entertainment, other). Currency: USD.
- **Time Tracker** - Track time spent on tasks/projects using start/stop timers or manual entry.
- **Exercise Tracker** - Log workouts by type (cardio, strength, flexibility, sports). Track duration, distance, sets/reps/weight.

## Project level Tech Decisions

- Language: Python (stdlib only, no external dependencies)
- Interface: HTML with interactive menus
- Storage: JSON files (stored alongside scripts)
